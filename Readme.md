Welcome the the Cool Events site!
In order to start the site run:

`npm i`

`npm start`

open at http://localhost:5000

## Lint and Format

Run lint checks:

`npm run lint`

Auto-fix lint issues:

`npm run lint:fix`

Run formatter:

`npm run format`

Check formatter without writing changes:

`npm run format:check`

Formatting uses LF line endings configured in `.prettierrc`.
